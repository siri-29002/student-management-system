## Description
A Spring Boot RESTful web application to manage student details such as name, email, and address. 
Provides CRUD operations using REST APIs and persists data in MySQL database.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- REST APIs
- MySQL
- Maven
- Git & GitHub
- Postman

## Features
- Create a new student
- Get student by ID
- Get all students
- Update student details
- Delete student by ID

## API Endpoints
- POST /api/students
- GET /api/students/{id}
- GET /api/students
- PUT /api/students/{id}
- DELETE /api/students/{id}

- Spring Tool Suite (STS)
- Postman
- MySQL Workbench
